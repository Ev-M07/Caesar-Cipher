Caesar Cipher Encryptor

the Caesar Cipher Works The Caesar cipher is a simple way to hide a message by shifting each letter a certain number of places in the alphabet.
Imagine the alphabet is a circle, and you move each letter forward by a set number. For example, if you shift by 3:
A becomes D
B becomes E
C becomes F
and so on, until Z, which wraps around to C.
So, the word "HELLO" with a shift of 3 becomes "KHOOR". To read the secret message, you just shift the letters back by the same number.
The Caesar cipher was used by Julius Caesar to keep his military messages secret.
It is easy to use but not very secure today because there are only 25 possible shifts, so someone can try them all quickly to decode the message.
